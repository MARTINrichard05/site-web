Nextcloud

Pour pouvoir synchroniser mes fichiers et les sauvegarder, j'ai mis en place un serveur nextcloud, de cette manière, je garde le controle de mes données et de ma vie privée.

Mon but étant de l'ouvrir a l'internet, il est donc nécessaire qu'il soir sécurisé, c'est pour cela que j'ai opté pour la version préconfigurée sur le store snapcraft qui est donc sécurisée par défaut (à condition de mettre un mdp fort).

Je n'ai pas ouvert le port 22(accès sécurisé a distance) pour éviter des attaques par bruteforce a distance, j'ai donc activé la fonction serveur OpenVpn sur mon routeur TpLink ce qui me permet d'y avoir un accès complet sécurisé a distance

Actuellement, je ne sais pas si je vais garder ce serveur indéfiniment, mais il est actuellement très utile, si je m'en sert plus je le reconvertirai en serveur home-assistant (plus performant que le serveur actuel pour la commande vocale)

Pour vérifier que le serveur est sécurisé, on peut utiliser l'outil fournit par nextcloud. Je l'ai donc fait et vous pouvez voir le resultat ci dessous (vous pouvez le faire si vous voulez vérifier)
